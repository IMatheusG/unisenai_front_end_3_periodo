Nome do projeto: Porfólio inicial

Objetivo: Apenas um portfólio simples para ter uma base de como fazer melhor depois.

Tecnologias utilizadas: html, css e js, usei uma biblioteca de JS chamada sweet alert para mostrar o alerta no canto da tela após o envio do formulário

Estrutura das páginas/seções: só tem a index, dividida por <section> para cada seção do portfólio

Funcionalidades implementadas: responsividade, tratamento de evento de carregamento do DOM, formulário para simular envio e exibição de mensagem enviada (sweet alert) e validação nos campos do formulário para total preenchimento.
